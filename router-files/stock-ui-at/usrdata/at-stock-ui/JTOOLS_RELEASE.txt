Jtools stock UI package
Release date: 2026-03-20

Current proven model
- Keep payload in /usrdata/at-stock-ui
- Build live trees in /usrdata/at-stock-ui/live
- Bind-mount live/www onto /www
- Bind-mount live/usr/share/lua/5.1/webif onto /usr/share/lua/5.1/webif
- Restart turbontc.service after binding
- Reapply late with jtools-stock-ui.timer
