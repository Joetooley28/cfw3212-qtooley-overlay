# Stock UI Fallback Package

This folder contains a public fallback stock web package for the Casa Systems `CFW-3212`.

Source:

- clean post-reset stock candidate capture
- router firmware: `USC_1.1.99.0`
- module firmware observed in metadata: `RG520NNADAR03A03M4G`

Included files:

- `stock-ui-fallback-USC_1.1.99.0.zip`
- `stock_www.tar`
- `stock_webif.tar`
- `SHA256SUMS.txt`

Use:

- emergency fallback stock web recovery only
- not a claim of exact per-device factory restoration
- prefer the router's own first-install baseline snapshot whenever available

See:

- [`docs/fallback-stock-recovery.md`](../../../docs/fallback-stock-recovery.md)
